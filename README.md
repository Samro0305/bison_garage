# Bison Garage

A professional Flutter-based garage management and billing application.

## Features

### Invoice Management

* Create customer invoices
* Multiple service items per invoice
* GST calculation support
* Automatic invoice numbering

### PDF Invoice Generation

* Professional 2-page invoice layout
* Garage logo support
* Signature support
* Watermark branding
* Share PDF directly from app

### Dashboard

* Revenue analytics
* Revenue trend charts
* Total invoices overview
* Business insights

### History

* Search invoices
* View invoice details
* Re-download invoices
* Track customer history

### Settings

* Garage information management
* Logo upload
* Signature upload
* Password protection

### Backup & Restore

* Export application data
* Restore backups
* Local data protection
